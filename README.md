# Project Lessons Learned

- JavaScript
- HTML
- CSS

## Instruções de Instalação e Uso

1. Clone este repositório para o seu computador:

    ```
    git@github.com:JyojiTenguam/ColorFlipper.git
    ```

2. Navegue até o diretório do projeto:

    ```
    cd ColorFlipper
    ```
